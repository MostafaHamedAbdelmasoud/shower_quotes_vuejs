## Project setup

## after cloning this project:

run:

```bash
cd project_code
```



## then run
```bash
npm install
```

## Compiles and hot-reloads for development:
```bash
npm run serve
```
## Compiles and minifies for production
```bash
npm run build
```
## Lints and fixes files
```bash
npm run lint
```
