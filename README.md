# shower_quotes_vuejs
it's a website which show some shower thoughts from reddit.com through api 

## vue-showerthoughts

##Project setup
```bash
npm install
```

## Compiles and hot-reloads for development:
```bash
npm run serve
```
## Compiles and minifies for production
```bash
npm run build
```
## Lints and fixes files

npm run lint
